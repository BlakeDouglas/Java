CSV to VCF converter using a report directly generated from the Verizon Portal. 

Create a custom report containing username and wireless number
Compile the Program using javac CsvToVcfConverter.java
Run the Program using java CsvToVcfConverter
Enter the path to your generated CSV file: i.e. C:\Users\JSmith\Documents\contacts.csv

The program will do the rest of the filtering and formatting -> providing a VCF file in the same directory the java file was saved
